Fundamentos

🧠O que é o Octave

  • Ambiente de cálculo numérico (praticamente igual ao MATLAB), feito para trabalhar com vetores e matrizes com poucas linhas.
  • Você pode digitar direto na Janela de Comandos (testar na hora) ou salvar um arquivo .m (um "script") e executar.
  • % inicia um comentário — o Octave ignora até o fim da linha.
  • ; no fim da linha esconde a saída; sem ; o Octave mostra o resultado na tela.
  • Para rodar um script, escreva o nome do arquivo (sem o .m) e aperte Enter.
% meu primeiro script x = 5; % ; esconde a saida y = 3 % sem ; -> mostra y = 3 disp('Ola, Octave!')
Ato 02 · Entrada e Saída

⌨️Ler e Escrever na tela

  • input lê um valor do teclado e guarda numa variável.
  • disp mostra um texto ou o valor de uma variável.
  • printf/fprintf mostram com formato: %d inteiro, %f real, %.2f 2 casas, \n pula linha, \t tabula.
  • sprintf funciona igual, mas em vez de mostrar, guarda o texto numa variável.
% litros de gasolina (Ato02) preco = input('Preco do litro: '); valor = input('Quanto abastecer: '); litros = valor / preco; printf('Voce abastece %.2f litros\n', litros)

Dica: o %.2f junto com a variável no fim do printf é o que formata o número com 2 casas decimais.

Operadores

➗Contas, Comparações e Lógica

  • Aritméticos: + − * /, potência ^, e o resto da divisão mod(a,b).
  • Relacionais (dão verdadeiro/falso): == igual, ~= diferente, < > <= >=.
  • Lógicos: && (E), || (OU), ! (NÃO) — para juntar condições.
  • ⚠️ Atenção: = atribui valor; == compara. São coisas diferentes!
  • Truque do par/ímpar: mod(x,2)==0 → é par.
% media ponderada (Ato02) n1=input('Nota1: '); n2=input('Nota2: '); n3=input('Nota3: '); mp = (2*n1 + 3*n2 + 5*n3) / 10; printf('Media = %.1f\n', mp)
Ato 03 · Decisões

🔀if / elseif / else

  • Executa um bloco somente se a condição for verdadeira.
  • Todo if precisa de um end no final (erro clássico esquecer!).
  • Vários casos em sequência? Use elseif; e else para "todo o resto".
  • O Octave testa de cima para baixo e para no primeiro que der verdadeiro.
% categoria do nadador (Ato03) idade = input('Idade: '); if idade >= 5 && idade <= 7 disp('Infantil') elseif idade <= 11 disp('Juvenil') elseif idade <= 17 disp('Adolescente') else disp('Adulto/Senior') end
Ato 04 · Repetição

🔁Laço for (repete uma quantidade certa)

  • for i = 1:n repete o bloco de 1 até n, somando 1 a cada volta.
  • Com passo: for i = 0:2:10 vai de 2 em 2 (0,2,4,…,10).
  • Fecha com end. É o laço ideal para percorrer vetores e matrizes.
  • A variável i é o contador — você pode usá-la dentro do laço.
% tabuada de um numero (Ato04) n = input('Numero: '); for i = 1:10 printf('%d x %d = %d\n', n, i, n*i) end
Numero: 3 3 x 1 = 3 3 x 2 = 6 3 x 3 = 9 ... 3 x 10 = 30
Ato 04 · Repetição

♾️Laço while (repete enquanto…)

  • Repete enquanto a condição for verdadeira — bom quando não se sabe quantas vezes vai repetir.
  • Algo dentro do laço precisa mudar a condição, senão ele trava (laço infinito!).
  • Uso clássico: validar entrada (só aceitar valor positivo, por exemplo).
  • for = número de vezes conhecido · while = "até acontecer algo".
% soma de 5 numeros positivos (Ato04) soma = 0; for i = 1:5 x = input('Positivo: '); while x < 0 % rejeita negativo x = input('Invalido, de novo: '); end soma = soma + x; end printf('Soma = %d\n', soma)
Vetores

📏Vetores (uma lista de números)

  • Criar: v = [10 20 30] · sequência 1:10 · zerado zeros(1,10).
  • Acessar uma posição: v(1) é o 1º. ⚠️ No Octave o índice começa em 1 (não em 0!).
  • Tamanho: length(v). Para preencher/ler, percorra com um for.
  • Funções prontas: max(v), min(v), sum(v), sort(v).
% vetor: dobro do indice (Vetores) for i = 1:20 v(i) = i * 2; % v(3) = 6, v(4) = 8... end % maior e menor de um vetor printf('Maior=%d Menor=%d\n', max(v), min(v))
Matrizes

🔢Matrizes (tabela linhas × colunas)

  • Criar: M = [1 2; 3 4] — o ; troca de linha · zerada zeros(5,5).
  • Elemento: M(i,j) = linha i, coluna j (sempre linha primeiro).
  • Para ler/percorrer, use 2 for aninhados: um roda as linhas, o de dentro as colunas.
  • Diagonal principal: i==j · Diagonal secundária: i+j==n+1.
% ler matriz 5x5 (2 lacos aninhados) for i = 1:5 for j = 1:5 M(i,j) = input(' '); end end M * 2 % multiplica a matriz toda por 2
Exemplo resolvido 1/5

✅Contar Pares e Ímpares (matriz)

Pares=0; Impares=0; for I=1:5 for J=1:3 if mod(A(I,J),2) == 0 Pares++; else Impares++; end end end printf('%d Pares e %d Impares\n',Pares,Impares)

💡 A ideia central: percorrer todos os elementos com 2 laços e testar mod(x,2)==0. Se o resto da divisão por 2 é zero, o número é par; senão, ímpar.

Pares++ é o mesmo que Pares = Pares + 1.

✳️Diagonal principal · Minimax

% diagonal principal = 1, resto = 0 R = zeros(5,5); for i=1:5 R(i,i) = 1; % i==j e a diagonal end
% minimax: menor da linha do maior [mx, idx] = max(M(:)); [lin, ~] = ind2sub(size(M), idx); mini = min(M(lin,:)); printf('Minimax=%d na linha %d\n',mini,lin)

Minimax = acha o maior da matriz, vê em que linha ele está, e pega o menor dessa linha. Nosso questao1 / questao6

🧮Matriz × valor → vetor · Maior da linha

% M(4x4)*A guardado num vetor V(16) K = 1; for I=1:4 for J=1:4 V(K) = M(I,J) * A; K++; % anda 1 no vetor end end
% maior numero de cada linha for i=1:5 maior = M(i,1); for j=2:4 if M(i,j) > maior maior = M(i,j); end end printf('Linha %d: %d\n',i,maior) end

Antes da prova

⚠️Erros comuns & Dicas rápidas

  • Índice começa em 1: v(1) é o primeiro. Não existe v(0).
  • Feche os blocos: todo if, for e while precisa de um end.
  • = vs ==: um atribui, o outro compara. Trocar dá erro ou resultado errado.
  • Ponto e vírgula: use ; dentro de laços para não poluir a tela de saída.
  • while infinito: garanta que algo muda a condição dentro do laço.
  • Matriz: é sempre M(linha, coluna) — nessa ordem.

Checklist mental: declarei/zerei as variáveis? · o laço cobre todos os elementos? · fechei todos os end? · o printf tem a variável no final?
