# ♻️ Sustenta ou Descarta

Jogo educacional de **Scratch (.sb3)** sobre **reciclagem**. O jogador observa um
item e decide se ele é **RECICLÁVEL** ou **DESCARTÁVEL (lixo comum)**, aprendendo
com dicas a cada resposta.

## 🎮 Como jogar
1. Abra o arquivo **`Sustenta_ou_Descarta.sb3`**.
2. Na tela inicial, pressione **ENTER** (ou clique no botão "Pressione Enter").
3. A cada rodada aparece um item no centro. Clique em:
   - **RECICLAR** (botão verde) se o item for reciclável;
   - **DESCARTAR** (botão marrom) se for lixo comum/rejeito.
4. Você tem **3 vidas** e um **cronômetro** por item. Erros e tempo esgotado tiram vida.
5. Vença as **7 fases** (a dificuldade e a velocidade aumentam) para virar um
   **Mestre da Reciclagem**!

## ▶️ Onde abrir
- **Online:** https://scratch.mit.edu/ → *Criar* → menu *Arquivo* → *Carregar do seu computador*.
- **Online (alternativa):** https://turbowarp.org/ → *Carregar projeto*.
- **Offline:** aplicativo **Scratch Desktop** → *Arquivo* → *Carregar do seu computador*.

## ✨ O que há no jogo
- **Tela inicial imersiva** com título, mascote **Eco**, painel de instruções,
  botão "Pressione Enter" animado e música de fundo.
- **7 fases** com 20 itens diferentes (recicláveis e descartáveis).
- **Itens recicláveis:** garrafa PET, lata de alumínio, jornal, caixa de papelão,
  garrafa de vidro, lata de conserva, caderno, garrafa térmica, sacola de pano,
  frasco de shampoo.
- **Itens descartáveis:** copo descartável, canudo, embalagem metalizada, sacola
  plástica, bituca de cigarro, guardanapo sujo, chiclete, fralda, isopor, esponja.
- **Feedback educativo** do mascote a cada resposta, **placar**, **vidas em corações**
  e **cronômetro** que diminui.
- **Telas de vitória e derrota** com opção de jogar novamente (ENTER).
