# Biblioteca de Variáveis Reutilizáveis
## Blocos de contexto para colar nos prompts

Preencha uma vez, reutilize sempre. Mantenha esta nota atualizada — ela é o seu "perfil" para IAs.

### Meu perfil profissional
```
About me: {profissão/estudos, nível de experiência, ferramentas que domina}
My goals: {objetivos atuais}
My constraints: {tempo disponível, orçamento, equipamentos}
```

### Meu projeto atual
```
Project: {nome e descrição em 2 frases}
Stage: {ideia | protótipo | produção}
Stack/tools: {tecnologias e ferramentas}
Audience: {para quem}
```

### Minha marca/estilo
```
Brand voice: {3 adjetivos}
Visual style: {referências, cores, o que evitar}
Language: {pt-BR para conteúdo local, en para código/docs}
```

### Preferências de resposta
```
Response style: direct, no filler, examples over theory.
Code style: complete runnable files, comments only where non-obvious.
When unsure, say so explicitly instead of guessing.
```

> Dica: no Claude/ChatGPT, salve estes blocos nas instruções personalizadas/projeto para não colar toda vez.
