# Fluxo de Trabalho: IA + Obsidian como Segundo Cérebro
## O ciclo do segundo cérebro com IA

1. **Capturar** — ideias, artigos, transcrições entram como notas rápidas.
2. **Processar com IA** — use o prompt `Second Brain AI Librarian` (categoria IA) para resumir em notas atômicas, sugerir links e extrair ações.
3. **Conectar** — linke as notas novas aos MOCs e conceitos existentes.
4. **Criar** — quando for produzir (texto, código, design), busque o prompt certo neste cofre e alimente-o com suas notas como contexto.
5. **Revisar** — semanalmente, use o prompt `Weekly Review Facilitator` (Produtividade).

## Levando o cofre para dentro da IA

- **Claude Projects / Custom GPTs**: suba suas notas-chave como conhecimento do projeto e cole o prompt do cofre como instrução.
- **Copiar contexto**: ao usar um prompt, cole também as notas relevantes do seu vault no campo de contexto — o resultado melhora dramaticamente.
- **MCP (Model Context Protocol)**: com um servidor MCP de sistema de arquivos apontando para o vault, o Claude lê e escreve suas notas diretamente. Veja o prompt `MCP Server Builder` para criar conectores personalizados.

## Automações que valem a pena

- Template + hotkey para nova nota de prompt.
- Plugin *Templater* para preencher data/categoria automaticamente.
- Plugin *Dataview* (opcional) para tabelas dinâmicas: todos os prompts por tag/dificuldade.
