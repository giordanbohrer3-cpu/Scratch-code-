# Como Usar Este Cofre
## O que é este cofre

Este é o seu **Segundo Cérebro de Prompts**: mais de mil prompts profissionais, cada um em seu próprio arquivo, organizados para o Obsidian. Todos os prompts estão em **inglês** (melhor desempenho na maioria dos modelos de IA), com instruções de uso em português.

## Estrutura

- **Memória Principal/** — todos os prompts, organizados em categorias numeradas (01-29).
- **Pontos Essenciais/** — guias fundamentais: engenharia de prompts, frameworks de pensamento, como escolher modelos.
- **Templates/** — modelos para criar seus próprios prompts e notas.
- **00 · Índice Geral** — o mapa de tudo (MOC).

## Como encontrar um prompt

1. **Busca** (`Ctrl/Cmd + Shift + F`): procure por palavra-chave, ex. `landing page`, `debugging`, `logo`.
2. **Tags**: cada prompt tem tags no frontmatter. Clique em uma tag ou use o painel de tags. Ex.: `#prompt/programming`, `#prompt/video`.
3. **Índices (MOCs)**: cada categoria tem um índice com todos os prompts listados.
4. **Graph view**: os índices conectam tudo — navegue visualmente.

## Como usar um prompt

1. Abra o arquivo do prompt.
2. Copie o bloco de código na seção **Prompt**.
3. Substitua os campos `{ENTRE_CHAVES}` pelos seus dados (a tabela **Variáveis** explica cada um).
4. Cole na IA (Claude, ChatGPT, Gemini...).
5. Responda às perguntas de esclarecimento que a IA fizer — é assim que o resultado fica excelente.

## Como expandir o cofre

- Use o template em `Templates/Template de Prompt.md` para criar novos prompts no mesmo padrão.
- Quando um prompt funcionar muito bem para você, adicione a tag `#favorito` e anote o resultado na própria nota.
- Crie MOCs pessoais ligando os prompts que você mais usa por projeto.
