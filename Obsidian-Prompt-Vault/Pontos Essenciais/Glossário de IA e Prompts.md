# Glossário de IA e Prompts
| Termo | Significado |
|---|---|
| **LLM** | Large Language Model — o motor por trás de Claude, GPT, Gemini |
| **Prompt** | A instrução/mensagem enviada à IA |
| **System prompt** | Instrução permanente que define comportamento (em projetos/GPTs) |
| **Token** | Unidade de texto (~4 caracteres); base de custo e limites |
| **Janela de contexto** | Quanto o modelo "lembra" na conversa atual |
| **Temperatura** | Aleatoriedade da resposta (baixa = consistente, alta = criativa) |
| **Few-shot** | Dar exemplos no prompt para ensinar o padrão |
| **Zero-shot** | Pedir sem exemplos |
| **Chain-of-thought** | Fazer o modelo raciocinar passo a passo antes de responder |
| **Alucinação** | Quando a IA inventa fatos com confiança |
| **RAG** | Retrieval-Augmented Generation — buscar em documentos antes de responder |
| **Embedding** | Representação numérica de texto para busca semântica |
| **Fine-tuning** | Retreinar um modelo com seus dados |
| **MCP** | Model Context Protocol — padrão para conectar IAs a ferramentas e dados |
| **Agente** | IA que usa ferramentas e executa múltiplos passos sozinha |
| **MOC** | Map of Content — nota-índice que organiza outras notas no Obsidian |
| **Zettelkasten** | Método de notas atômicas interligadas |
| **PARA** | Projects, Areas, Resources, Archive — método de organização |
