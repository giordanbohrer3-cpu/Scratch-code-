Fundamentos

🧠O que é o Octave

  • Ambiente de cálculo numérico (quase igual ao MATLAB) — ótimo para vetores e matrizes.
  • Você digita na Janela de Comandos ou salva um arquivo .m e executa.
  • % começa um comentário.
  • ; no fim da linha esconde a saída; sem ; ele mostra o resultado.
  • Rodar um script: escreva o nome do arquivo (sem .m) e Enter.
% meu primeiro script x = 5; % ; esconde a saida y = 3 % sem ; -> mostra y = 3 disp('Ola, Octave!')
Ato 02 · Entrada e Saída

⌨️Ler e Escrever

  • input lê um valor do teclado.
  • disp mostra um texto ou valor.
  • printf/fprintf mostram com formato: %d inteiro, %f real, \n nova linha, \t tabulação.
  • sprintf monta um texto formatado numa variável.
% media de gasolina (Ato02) preco = input('Preco do litro: '); valor = input('Quanto abastecer: '); litros = valor / preco; printf('Voce abastece %.2f litros\n', litros)
Operadores

➗Contas, Comparações e Lógica

  • Aritméticos: + − * /, potência ^, resto da divisão mod(a,b).
  • Relacionais: == igual, ~= diferente, < > <= >=.
  • Lógicos: && (e), || (ou), ! (não).
  • Truque do par/ímpar: mod(x,2)==0 → par.
% media ponderada (Ato02) n1=input('Nota1: '); n2=input('Nota2: '); n3=input('Nota3: '); mp = (2*n1 + 3*n2 + 5*n3) / 10; printf('Media = %.1f\n', mp)
Ato 03 · Decisões

🔀if / elseif / else

  • Executa um bloco só se a condição for verdadeira.
  • Sempre fecha com end.
  • Vários casos? Use elseif em cadeia.
% categoria do nadador (Ato03) idade = input('Idade: '); if idade >= 5 && idade <= 7 disp('Infantil') elseif idade <= 11 disp('Juvenil') elseif idade <= 17 disp('Adolescente') else disp('Adulto/Senior') end
Ato 04 · Repetição

🔁Laço for

  • for i = 1:n repete de 1 até n.
  • Pode ter passo: for i = 0:2:10 (de 2 em 2).
  • Fecha com end. Ideal para percorrer vetores/matrizes.
% tabuada de um numero (Ato04) n = input('Numero: '); for i = 1:10 printf('%d x %d = %d\n', n, i, n*i) end
Numero: 3 3 x 1 = 3 3 x 2 = 6 3 x 3 = 9 ... 3 x 10 = 30
Ato 04 · Repetição

♾️Laço while (repete enquanto…)

  • Repete enquanto a condição for verdadeira — bom quando não se sabe quantas vezes.
  • Cuidado para a condição mudar dentro do laço (senão trava!).
  • Clássico: validar entrada (só aceitar positivo).
% soma de 5 numeros positivos (Ato04) soma = 0; for i = 1:5 x = input('Positivo: '); while x < 0 % rejeita negativo x = input('Invalido, de novo: '); end soma = soma + x; end printf('Soma = %d\n', soma)
Vetores

📏Vetores (lista de números)

  • Criar: v = [10 20 30] · sequência 1:10 · zerado zeros(1,10).
  • Acessar posição: v(1) é o 1º (índice começa em 1!).
  • Tamanho: length(v). Percorrer com for.
  • Prontos: max(v), min(v), sum(v), sort(v).
% vetor: dobro do indice (Vetores) for i = 1:20 v(i) = i * 2; end % maior e menor de um vetor printf('Maior=%d Menor=%d\n', max(v), min(v))
Matrizes

🔢Matrizes (tabela linhas × colunas)

  • Criar: M = [1 2; 3 4] (; troca de linha) · zeros(5,5).
  • Elemento: M(i,j) = linha i, coluna j.
  • Ler com 2 for aninhados (um por linha, um por coluna).
  • Diagonal principal: i==j · secundária: i+j==n+1.
% ler matriz 5x5 (2 laços) for i = 1:5 for j = 1:5 M(i,j) = input(' '); end end M * 2 % multiplica tudo por 2
Exemplo resolvido 1/5

✅Contar Pares e Ímpares (matriz)

Pares=0; Impares=0; for I=1:5 for J=1:3 if mod(A(I,J),2) == 0 Pares++; else Impares++; end end end printf('%d Pares e %d Impares\n',Pares,Impares)

💡 A chave é mod(x,2)==0: se o resto por 2 é zero, é par.
