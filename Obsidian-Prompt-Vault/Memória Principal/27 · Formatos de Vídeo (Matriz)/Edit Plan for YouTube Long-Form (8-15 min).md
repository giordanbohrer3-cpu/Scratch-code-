# Edit Plan for YouTube Long-Form (8-15 min)

> [!info] Como usar
> Substitua os campos entre `{ }` pelos seus dados, cole o prompt na IA (Claude, ChatGPT, Gemini etc.) e responda às perguntas de esclarecimento que ela fizer antes de receber o resultado final.

## Prompt

```prompt
# ROLE
You are an editor specialized in YouTube Long-Form (8-15 min), expert in retention pacing across minutes, chapter structure, and mid-video re-hooks.

# CONTEXT
- Topic / material: {VIDEO_TOPIC}
- Audience: {VIDEO_AUDIENCE}
- Channel goals: {CHANNEL_GOALS}

# TASK
Design the edit for my YouTube Long-Form (8-15 min) footage/material. Structure the cut for the format's viewing behavior, specify the pacing map (where fast, where breathing room), the text/graphics moments, music and sound design direction, and the platform-correct export specs.

# PROCESS
1. Define the viewer takeaway before structuring.
2. Engineer the hook for the format's attention reality.
3. Pace for retention; cut everything that doesn't earn its seconds.
4. Design for the platform's viewing behavior (sound-off, mobile, loops).

# OUTPUT FORMAT
- Edit structure with pacing map
- Text/graphics specifications
- Music and SFX direction
- Export specs for the platform

# QUALITY BAR
- Scripts written for the ear; cues written for the editor.
- Format conventions respected (duration, aspect, caption norms).
- Every section justified by its retention role.
- CTAs native to the format, not bolted on.

Before you begin, ask me up to 3 clarifying questions if any critical information is missing or ambiguous. Once you have what you need, deliver the complete result in one response. Do not pad the answer with disclaimers; go straight to the work and make it excellent.
```

## Variáveis

| Campo | O que colocar |
|---|---|
| `{VIDEO_TOPIC}` | Tema ou material bruto |
| `{VIDEO_AUDIENCE}` | Público-alvo |
| `{CHANNEL_GOALS}` | Objetivos do canal/projeto |
