# Taxonomia de Tags do Cofre
## Como as tags funcionam neste cofre

Todo prompt tem no frontmatter:

- `prompt` — tag universal (todo prompt tem).
- Tag da **categoria**: `programming`, `web-development`, `app-development`, `game-development`, `ai`, `data-science`, `devops`, `security`, `architecture`, `graphic-design`, `video`, `photography`, `ui-ux`, `writing`, `music`, `marketing`, `business`, `career`, `education`, `productivity`, `finance`, `life`, `research`...
- Tags de **assunto**: `react`, `logo`, `debugging`, `midjourney`, `seo`, `obsidian`...
- Campo `difficulty`: `beginner` | `intermediate` | `advanced`.
- Campo `source`: `original` (criado para este cofre) | `awesome-chatgpt-prompts` | `llm-prompt-library`.

## Buscas úteis

- Todos os prompts de uma área: `tag:#programming`
- Por dificuldade (busca): `["difficulty": "beginner"]` ou busque `difficulty: beginner`
- Prompts de imagem: `tag:#image-generation`
- Combinações: `tag:#game-development tag:#scratch`

## Tags pessoais sugeridas (adicione você mesmo)

- `#favorito` — prompts que funcionaram muito bem para você
- `#testado` — já usados com sucesso
- `#projeto/nome-do-projeto` — prompts usados em cada projeto seu
