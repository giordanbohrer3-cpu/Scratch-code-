# Trabalho Octave – Matrizes

Trabalho da disciplina **Programação Algorítmica** (Prof. Jonas Schreiber – UNIJUÍ),
valendo 30 pontos. Contém a resolução, em Octave, das 6 questões sobre **matrizes**.

## Arquivos

| Arquivo | Questão |
|---------|---------|
| `questao1.m` | Matriz 5x5: preenche a diagonal **principal** com 1 e o resto com 0. |
| `questao2.m` | Matriz 5x5: preenche a diagonal **secundária** com 2 e o resto com 0. |
| `questao3.m` | Matriz 6x6: calcula a **soma** da diagonal secundária. |
| `questao4.m` | Matriz 4x3: conta **pares e ímpares** e mostra o percentual. |
| `questao5.m` | Matriz 5x4: mostra o **maior número de cada linha**. |
| `questao6.m` | Matriz 4x4: encontra o elemento **minimax** e sua posição. |
| `Trabalho_Octave_Matrizes.pdf` | PDF com o enunciado, o código e a saída de exemplo de cada questão. |

## Como executar

Cada programa lê a matriz **elemento a elemento** pelo teclado. No Octave:

```octave
questao1
```

ou, pelo terminal:

```bash
octave questao1.m
```

Digite os valores conforme solicitado (um por vez) e o programa mostra o resultado.
