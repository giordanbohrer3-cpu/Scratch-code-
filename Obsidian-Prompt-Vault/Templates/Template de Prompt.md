## Prompt

```prompt
# ROLE
You are {expert persona with specific experience}.

# CONTEXT
- Project: {PROJECT}
- Goal: {GOAL}
- Constraints: {CONSTRAINTS}

# TASK
{The specific mission, one clear verb.}

# PROCESS
1. {Step one}
2. {Step two}
3. {Step three}

# OUTPUT FORMAT
- {Deliverable 1}
- {Deliverable 2}

# QUALITY BAR
- {What separates excellent from acceptable}

Before you begin, ask me up to 3 clarifying questions if any critical information is missing.
```
